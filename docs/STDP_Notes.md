STDP = Spike-Timing Dependent Plasticity
       It is a learning rule for synapses

        |           Case	       |         Result          |
        |   Pre spikes BEFORE Post | Weight increases (LTP)  |
        |   Post spikes BEFORE Pre | Weight decreases (LTD)  |


🧠 Intuition:-
    If neuron A fires and causes neuron B to fire   → strengthen connection.
    If neuron B fires before A                      → weaken connection.
